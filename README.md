### **2. XSS Attack simulation (XSS Attack)**
**File:** `XSS-Attack.pdf`  
A complete walkthrough of exploiting and mitigating Cross-Site Scripting (XSS) vulnerabilities in the Elgg social networking system.

**Covered Tasks**
- Basic XSS injection (`<script>alert("XSS")</script>`)
- Stored XSS in profile fields  
- Cookie stealing using JavaScript and Netcat  
- Samy-style self-replicating worm  
- Automatic friend-adding worm  
- DOM-based XSS explanation  
- Restoring server-side protections using `htmlspecialchars()`  

**Skills Demonstrated**
- Reflected, stored, and DOM-based XSS  
- Session hijacking through cookie exfiltration  
- Crafting malicious JavaScript payloads  
- Building self-propagating worms  
- Understanding CSRF-like behaviors with Elgg tokens  
- Hardening applications using output encoding and sanitization
